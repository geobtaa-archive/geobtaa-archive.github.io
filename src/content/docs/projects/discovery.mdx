<h2>GeoBTAA API</h2>

![](@images/strategic-tech-plan.png)

We are building an API that will serve as a central access point for sharing metadata records for geospatial resources. It will support integration with a variety of frontend applications and provide filtered metadata outputs for reuse. We will document its schema and endpoints to facilitate community adoption.

[<Icon name="external" class='inline-icon'/> Read more about the initiative](https://gin.btaa.org/ld/)
</Card>


<Card title="BTAA Geoportal">

![](@images/geoportal-homepage.png)

Our flagship project is a search portal that connects users to digital geospatial resources, including GIS datasets, web services, and digitized historical maps from multiple data clearinghouses and library catalogs. 

[<Icon name="external" class='inline-icon'/> Visit the Geoportal](https://geo.btaa.org)

[<Icon name="external" class='inline-icon'/> Read more about the Geoportal's key features and testimonials](geoportal.md).

</Card>
